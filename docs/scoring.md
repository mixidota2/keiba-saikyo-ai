# スコアリング基準

## 概要

各馬を8つのファクターで0〜100点評価し、重み付き合計で総合スコアを算出する。
総合スコアとオッズの比較により期待回収率を計算し、回収率最大化の買い目を作成する。

## ファクター別スコアリング詳細

### 1. 近走力 (recent_form) - デフォルト重み: 0.25

最も重要なファクター。直近のパフォーマンスを数値化する。

#### 前走スコアの計算

```
前走スコア = 基準タイム差 + 着差補正 + 上がり順位補正 + クラス補正
```

| 要素 | 計算方法 | 最大寄与 |
|------|---------|---------|
| 基準タイム差 | (基準タイム - 走破タイム) × 10 | ±30 |
| 着差補正 | 1着=+5, 2着=+3, 3着=+1, 4-5着=0, 6着〜=着差×(-1) | +5〜-20 |
| 上がり順位 | 1位=+3, 2位=+2, 3位=+1 | +3 |
| クラス補正 | G1=+10, G2=+7, G3=+5, OP=+3, 3勝=0, 2勝=-3, 1勝=-5 | ±10 |

#### 近走力スコアの算出（0〜100）

```
近走力スコア = min(100, max(0,
  前走スコア × 1.5        (最大40点)
  + 近5走平均スコア × 1.0  (最大30点)
  + 安定性スコア           (最大20点)
  + トレンドボーナス        (最大10点)
))
```

- **安定性スコア**: 近5走スコアの標準偏差が小さいほど高い
  - σ < 5: 20点, σ < 10: 15点, σ < 15: 10点, σ ≥ 15: 5点
- **トレンドボーナス**: 最近3走のスコアが上昇傾向なら+10

### 2. コース適性 (course_aptitude) - デフォルト重み: 0.20

```
コース適性スコア =
  当該コース成績スコア × 0.5   (最大50点)
  + 同系統コース成績 × 0.3      (最大30点)
  + タイム比較 × 0.2            (最大20点)
```

- 当該コース成績スコア = (勝率×40 + 連対率×30 + 複勝率×30) / 100 × 50
- 出走回数2回以下の場合: スコアを50点方向に30%補正（信頼度が低いため）

### 3. 距離適性 (distance_aptitude) - デフォルト重み: 0.15

```
距離適性スコア =
  当該距離成績 × 0.5          (最大50点)
  + 近似距離成績 × 0.25        (最大25点)
  + 血統適性推定 × 0.15        (最大15点)
  + レースパターン × 0.10      (最大10点)
```

### 4. 騎手力 (jockey) - デフォルト重み: 0.10

```
騎手スコア =
  全体成績 × 0.3              (最大30点)
  + コース成績 × 0.3           (最大30点)
  + 当該馬との相性 × 0.25      (最大25点)
  + 乗り替わり補正             (±15点)
```

- 乗り替わり補正: リーディング上位→下位=-10, 下位→上位=+10, 横ばい=0

### 5. 枠順 (draw) - デフォルト重み: 0.10

```
枠順スコア =
  コース枠番傾向 × 0.5        (最大50点)
  + バイアス補正 × 0.3         (最大30点)
  + 脚質相性 × 0.2            (最大20点)
```

### 6. 展開・ペース (pace) - デフォルト重み: 0.10

```
ペース予想:
- 逃げ馬2頭以上 → ハイペース傾向
- 逃げ馬1頭のみ → スロー〜ミドル
- 先行馬多数 → ミドル〜ハイ

ペーススコア:
- ハイペース時: 差し・追込 +20, 逃げ -20
- スローペース時: 逃げ・先行 +20, 追込 -20
- ミドルペース時: 全脚質 ±0
```

### 7. 馬場適性 (track_condition) - デフォルト重み: 0.05

```
馬場適性スコア =
  当該馬場状態での成績 × 0.6   (最大60点)
  + 血統馬場適性 × 0.25        (最大25点)
  + 推定補正 × 0.15            (最大15点)
```

### 8. クラス適性 (class_level) - デフォルト重み: 0.05

```
クラス適性スコア =
  同クラス成績 × 0.5           (最大50点)
  + クラス変動補正 × 0.3       (最大30点)
  + 相手関係 × 0.2             (最大20点)
```

## 総合スコアと期待回収率

### 総合スコア

```
総合スコア = Σ (ファクタースコア_i × 重み_i)
```

### 勝率推定

```
各馬の推定勝率 = (総合スコア^2) / Σ(全馬の総合スコア^2)
```

2乗を使うことで、上位馬と下位馬の差を適切に拡大する。

### 期待回収率

```
期待回収率(%) = 推定勝率 × 単勝オッズ × 100

判定:
- 150%以上: 強く推奨
- 120%以上: 推奨
- 100%以上: 買い検討
- 100%未満: 見送り
```

## ファクター重みの調整ガイドライン

レース特性に応じて重みを調整する。以下は目安。

| レース特性 | 調整内容 |
|-----------|---------|
| 短距離（1200m以下） | 枠順↑, ペース↑, 距離適性↓ |
| 長距離（2400m以上） | 近走力↑, クラス↑, 枠順↓ |
| 重・不良馬場 | 馬場適性↑↑, 枠順↑ |
| 小回りコース | 枠順↑, ペース↑, コース適性↑ |
| 広いコース | コース適性↓（差が出にくい）, 近走力↑ |
| 少頭数（8頭以下） | 枠順↓, ペース↑ |
| 多頭数（16頭以上） | 枠順↑, 騎手↑ |
