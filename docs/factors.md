# 予想ファクター一覧

## ファクター体系

```
予想ファクター
├── 主要ファクター（事前に評価可能）
│   ├── 近走力 (recent_form)
│   ├── コース適性 (course_aptitude)
│   ├── 距離適性 (distance_aptitude)
│   ├── 騎手力 (jockey)
│   ├── 枠順 (draw)
│   ├── 展開・ペース (pace)
│   ├── 馬場適性 (track_condition)
│   └── クラス適性 (class_level)
└── 当日補正ファクター（レース直前に適用）
    ├── 馬体重変動
    ├── トラックバイアス（当日）
    ├── パドック情報
    └── オッズ変動
```

## 主要ファクター詳細

### 近走力 (recent_form)
**何を見るか**: 直近5走のパフォーマンス

| 情報 | 重要度 | 取得方法 |
|------|--------|---------|
| 前走着順・着差 | 最重要 | Web検索 |
| 前走タイム | 最重要 | Web検索 |
| 上がり3Fタイム | 重要 | Web検索 |
| 前走の相手レベル | 重要 | Web検索 |
| 近走のスコア推移 | 重要 | 計算 |
| 休み明けかどうか | 参考 | Web検索 |

**注意点**:
- 前走が特殊条件（海外、地方）の場合はスコアを慎重に評価
- 長期休養明けは過去のパターンを参照
- 前走で不利があった場合は着順以上の評価をする

### コース適性 (course_aptitude)
**何を見るか**: 当該コースでの過去成績

| 情報 | 重要度 | 取得方法 |
|------|--------|---------|
| 当該コース成績 | 最重要 | Web検索 |
| 同系統コース成績 | 重要 | Web検索 |
| コース形態への適性 | 参考 | 推論 |

**コース分類**:
- 右回り vs 左回り
- 坂あり vs 平坦
- 小回り vs 大回り
- 直線距離

### 距離適性 (distance_aptitude)
**何を見るか**: 当該距離での適性

| 情報 | 重要度 | 取得方法 |
|------|--------|---------|
| 当該距離成績 | 最重要 | Web検索 |
| ±200m成績 | 重要 | Web検索 |
| 血統の距離適性 | 参考 | Web検索/推論 |
| 上がりタイムパターン | 参考 | 計算 |

**距離区分**:
- スプリント: 1000m〜1400m
- マイル: 1400m〜1800m
- 中距離: 1800m〜2200m
- 長距離: 2200m以上

### 騎手力 (jockey)
**何を見るか**: 騎手の能力と相性

| 情報 | 重要度 | 取得方法 |
|------|--------|---------|
| 騎手全体成績 | 重要 | Web検索 |
| 騎手コース成績 | 重要 | Web検索 |
| 当該馬との相性 | 重要 | Web検索 |
| 乗り替わり | 重要 | Web検索 |
| 騎手の得意パターン | 参考 | 推論 |

### 枠順 (draw)
**何を見るか**: 枠番の有利不利

| 情報 | 重要度 | 取得方法 |
|------|--------|---------|
| コース別枠番成績 | 重要 | Web検索 |
| 当日のバイアス | 重要 | Web検索（当日） |
| 脚質との組み合わせ | 重要 | 推論 |

**一般的な傾向**:
- 小回り: 内枠有利
- 直線長い: 枠の差は小さい
- 多頭数: 外枠不利が拡大
- ダート短距離: 先行力のある外枠も可

### 展開・ペース (pace)
**何を見るか**: レースのペース予想と脚質適性

| 情報 | 重要度 | 取得方法 |
|------|--------|---------|
| 各馬の脚質 | 最重要 | Web検索 |
| 逃げ馬の数 | 重要 | 分析 |
| 先行馬の数 | 重要 | 分析 |
| 過去の同条件ペース | 参考 | Web検索 |

**ペース判定基準**:
| 逃げ馬 | 先行馬 | ペース予想 |
|--------|--------|-----------|
| 0頭 | 0-2頭 | 超スロー |
| 1頭 | 0-2頭 | スロー |
| 1頭 | 3頭以上 | ミドル |
| 2頭 | - | ハイ |
| 3頭以上 | - | 超ハイ |

### 馬場適性 (track_condition)
**何を見るか**: 馬場状態への適性

| 情報 | 重要度 | 取得方法 |
|------|--------|---------|
| 馬場状態別成績 | 最重要 | Web検索 |
| 血統の馬場適性 | 重要 | Web検索/推論 |
| 当日の馬場状態 | 重要 | Web検索（当日） |

**重馬場巧者の血統傾向**:
- ロベルト系: 重馬場◎
- サンデーサイレンス系: 良馬場寄り
- ノーザンダンサー系: 重馬場○

### クラス適性 (class_level)
**何を見るか**: 現クラスでの実績と格

| 情報 | 重要度 | 取得方法 |
|------|--------|---------|
| 同クラス成績 | 重要 | Web検索 |
| 格上挑戦/格下げ | 重要 | 分析 |
| 対戦相手のレベル | 参考 | Web検索 |

## 当日補正ファクター

### 馬体重変動
レース直前に発表される馬体重による補正。

| 変動 | 補正 |
|------|------|
| ±0〜2kg | 補正なし |
| +3〜6kg | 要注意（太め残り？好調？）|
| +7kg以上 | 大幅減点（太め残り）|
| -3〜6kg | 要注意（絞れた？体調不良？）|
| -7kg以上 | 大幅減点（消耗の可能性）|

- 過去の好走時体重との比較が重要
- 成長期（3歳〜4歳）の増加はプラス材料の場合も

### トラックバイアス（当日）
当日の前レース結果から判定するバイアス。

| バイアス | 補正対象 |
|---------|---------|
| 内有利 | 内枠・先行馬に+補正 |
| 外有利 | 外枠・差し馬に+補正 |
| 前有利 | 先行脚質に+補正 |
| 後有利 | 差し・追込に+補正 |

### パドック情報
馬体チェックの情報（ユーザーからの追加入力）。

- 気配良好 → +5点
- 発汗多い → -5点（精神的に余裕なし）
- イレ込み → -10点
- 落ち着き → +3点

### オッズ変動
直前のオッズ変動による補正。

| 変動 | 解釈 |
|------|------|
| 急激なオッズ低下 | 関係者の自信？情報あり？ |
| 急激なオッズ上昇 | 状態面の不安？ |
| 安定 | 特に補正なし |

オッズ変動は参考情報とし、スコアへの直接反映は慎重に行う。
