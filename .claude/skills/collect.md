# /collect - 情報収集スキル

## 概要
指定レースの情報のみを収集し、永続化する。予想は行わない。

## 使い方
```
/collect 2026年2月15日 東京11R フェブラリーS
/collect 今週の重賞一覧
```

## 共通ルール
CLAUDE.md の「データ正確性の絶対ルール」「サブエージェントのツール使用ルール」を遵守すること。

## 実行フロー

### 1. レース特定
ユーザー指定のレースをWebSearchで特定する。

### 2. 情報収集
`agents/collect.md` のプロンプトに従い、Task toolでCollect Agentを起動する。
出走馬・枠順・オッズ・近走成績・持時計・基準タイム・トラックバイアス等を収集する。

### 3. 収集サマリー出力
収集した情報の概要を表形式で出力する:

```
## 収集完了: {レース名}

### 出走馬一覧
| 馬番 | 馬名 | 騎手 | 前走 | 単勝オッズ |
|------|------|------|------|-----------|
| ...  |      |      |      |           |

### 収集した追加情報
- トラックバイアス: ...
- 馬場状態: ...
- 注目ポイント: ...
```
